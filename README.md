\# Weather ETL Pipeline (Beginner Data Engineering Project)



This project implements an end-to-end ETL pipeline that ingests hourly weather data from the Open-Meteo API, performs transformations and data-quality checks using Python and pandas, and loads the data into a local SQLite database.



\## Tech Stack

\- Python

\- pandas, requests, sqlalchemy

\- SQLite

\- Conda environment

\- Windows Task Scheduler (for automation)



\## Pipeline Flow

API → Transform (pandas) → Quality Checks → SQLite
